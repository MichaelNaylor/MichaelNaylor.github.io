Information Systems Unit


About this unit


The Information Systems unit focusses more on group projects than solo work. In this unit we have been given a case studying for a bike rental shop that currently uses a paper based system. The assignment for this unit is group based and I am working on this assignment with 4 other students. For this unit we will go through the entire systems analysis and design process from understanding the case study to eventually creating the system in Oracle.


For the first part of this unit we produced a report on the case study identifying the issues with the current system and what the new system will need to do. We then explained the different trypes of data queries and management reports available and their use in business. Finally we outlined the specific data queries and management reports that the rental store will need.


For the next part of the project we need to start to design the system. To do this we are using licenced software called Visual Paradigm to create use case and entity relationsip diagrams. This part of the project will take the next few weeks to complete and will result in a clear design that we can use to create the system next term.
