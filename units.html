Unit Summary


This page contains a summary of each unit that I am currently studying. You can click on the heading of each unit to go to the page for that unit which will contain more information and examples of work completed



Programming

For the first year of my course I am focussing on the Java programming language. For the first few weeks we are using a graphical IDE called Processing. This program is available to download for free at processing.org.


Once we have become familiar with the core cencepts of the Java language using Processing we will move on to using a widely used professional IDE called Eclipse. This IDE is also available to download for free at eclipse.org.



Computer Systems Fundamentals

For the first term of the Computer Systems Fundametals unit we have 2 lectures each week. 1 lecture focusses on maths and one on computer logic and assembly language. For the first 7 weeks we are focussing on logic gates and boolean algebra to understand how the ALU works in modern computer systems before moving on to using Mars MIPS to create programs in assembly language.



Introduction to Web Development

In the Introduction to Web Development unit we focus on HTML, CSS and Javascript. The first few weeks have been focussed basic HTML and CSS before moving onto HTML 5 and CSS 3. From week 6 onwards we will also be learning Javascript and later in the year JQery. Part of the assignment for this unit is the creation of this website.



Information Systems

The Information Systems unit focusses more on group projects than solo work. In this unit we have been given a case studying for a bike rental shop that currently uses a paper based system. The assignment for this unit is group based and I am working on this assignment with 4 other students. For this unit we will go through the entire systems analysis and design process from understanding the case study to eventually creating the system in Oracle.
