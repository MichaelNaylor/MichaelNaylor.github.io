Introduction to Web Development Unit


About this unit


In the Introduction to Web Development unit we focus on HTML, CSS and Javascript. The first few weeks have been focussed basic HTML and CSS before moving onto HTML 5 and CSS 3. From week 6 onwards we will also be learning Javascript and later in the year JQery. Part of the assignment for this unit is the creation of this website.


Outside of the lectures of this course I have used various online resources to help me build this website. The website that I have used the most is W3 Schools. This website has tutorials on almost every aspect of HTML, CSS, Javascript and more. Each element has its own page with a description of what it does and examples of its use. Many of these pages also hava a try it yourself function where you can play around with the code and see the result on screen.
